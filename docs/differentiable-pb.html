Exact gradients through the nonlinear PB solve

An implicit-function adjoint for the converged Poisson–Boltzmann solve, including the dipole self-consistency loop, verified against finite differences on the production grid — with the forward solution bit-identical. Status: infrastructure behind a flag; training benefit not yet established.

1What has to be differentiated

The model's density coefficients c reach the solvent observables through a pipeline whose middle is an iterative solver. Everything outside the solve is ordinary autograd; the solve itself gets an implicit derivative:

Forward pipeline (solid) and the two loops that make it non-trivial: the fixsol dipole self-consistency (the solvent's own dipole feeds back into the source) and the backward path (dashed), which never re-enters the solver iterations — it solves one linear adjoint system at the converged solution.

2Why not backpropagate through the iterations

The gradient of a converged solution depends only on the solution and the operator — not on the path the solver took. Unrolling the Newton–PCG iterations is also numerically unstable: each CG step divides by residual inner products,

CG recurrences αk = ⟨rk, zk⟩ / ⟨pk, A pk⟩ ,  βk = ⟨rk+1, zk+1⟩ / ⟨rk, zk⟩

whose denominators vanish as CG converges; differentiating them multiplies the backward pass by their inverse squares. Measured: unrolled gradients reach 1056 once the iteration count grows (small, quickly-converging problems can appear to work — a trap).

3The implicit-function adjoint

Let p stand for the solver inputs (φ_sol, s_ion, s_diel), all differentiable functions of c. At the converged solution the residual vanishes identically in p, so its total derivative is zero:

implicit function theorem R(φ*(p), p) = 0  ⇒  J · dφ*/dp = − ∂R/∂p ,   J ≡ ∂R/∂φ |φ*

For a scalar loss L(φ*, p) one never forms dφ*/dp; a single adjoint solve suffices:

adjoint form JTλ = ∂L/∂φ*  ⇒  dL/dp = ∂L/∂p − (∂R/∂p)Tλ

Everything on the right is one linear solve plus vector–Jacobian products of the residual — evaluated in real space, which sidesteps the solver's half-spectrum and G=0 bookkeeping entirely. Measured properties of J (dense oracle on a small grid): symmetric to 10−16, negative definite, condition number ≈ 4.6×10². So the SPD system (−J)λ = −∂L/∂φ* is solved by preconditioned CG.

4Preconditioning — the charge mode is the trap

The residual's G=0 row is not a gauge freedom: it carries the charge-neutrality constraint, RG=0 = ⟨n_b+n_ion⟩ − q_sol, and couples to every charge-like input direction. The forward solver's inverse-Laplacian preconditioner vanishes exactly there, so plain PCG can never reduce the constant-mode residual — λ converges everywhere except in the one mode that charge-type gradients need. The fix is a one-number augmentation, computed with a single extra operator application:

preconditioner M−1 =  EDEPS / (4π²G²V)  ⊕  m₀₀ ,   m₀₀ = 1 / |⟨e, (−J)e⟩| ,   e = const.
Final adjoint relative residual (log scale) for four solver configurations on the production grid, with iteration counts. Without the constant-mode term the solve stalls — and the stalled component is exactly the charge mode, which is why monopole-coefficient gradients were the ones that came out wrong.

5The Jacobian is already in the solver

The Newton iteration linearizes the same residual, so its tangent operator Lop (dielectric tensor response + Boltzmann ion response + Laplacian) is the exact Jacobian, up to the FFT change of basis:

analytic Jacobian J v = − F−1 Lop F v

Probed against the autograd vector–Jacobian product on random vectors: relative difference 3×10−16. Applying it costs a few FFTs — about 100× less than rebuilding an autograd graph per CG iteration — and a one-time runtime self-check per grid guards the identity, falling back to the autograd operator on mismatch.

6Differentiating the dipole loop exactly

The solver runs a short fixed-point loop for the slab dipole correction: each step feeds the solvent dipole of the previous solution back into the source,

fixsol step k dipk = dval(ρ) + dsol(φ*k−1) − qsol zc ,  d̄k = d̄k−1 + α(rk) rk ,  cvdipk = Ĉ(z) d̄k

with rk = dipk − d̄k−1. The loop is finite (5 steps), so its exact derivative is the chain over the steps — replayed with linearizations anchored at the captured forward values, which are exact because every link (Ewald field, cdipol ramp, mixer update) is linear given the captured state. The one subtlety is the adaptive mixing gain: α is itself a function of the residual, and for |r| > 1 (α = α₀/r²) the true local derivative flips sign,

mixer gain — the last bug g ≡ ∂d̄k/∂dipk = ∂[α(r)r]/∂r = { +α  (|r| ≤ 1) ;  −α  (|r| > 1) }
The mixer's local gain g(r) (blue) versus the naive assumption g = α (yellow, dashed). For |r| > 1 the true derivative is negative; using +α there flipped the sign of the whole dipole-feedback gradient — reproducing exactly the observed symmetric error (+0.058 computed vs −0.064 true). Freezing the loop instead is not acceptable: the feedback flips the sign of charge-mode gradients.

7Verification

Central finite differences on the production setup (100×100×300 grid, real DFT baseline cache, real structure, forward tolerance 10−8). A like-for-like harness freezes the dipole correction in both the gradient and the FD probe to test the adjoint core in isolation; the full chain is then tested against unfrozen FD. Two figures carry the result:

Monopole-coefficient gradient error |ratio − 1| (log scale) at each stage of the campaign. Two structural fixes carry almost all of it: the constant-mode preconditioner term (adjoint core → exact) and the mixer's local derivative (full chain → exact).
Parity: every verified gradient against its finite-difference value (log–log, magnitudes spanning seven orders). All points sit on the diagonal; worst deviation 2%. Forward output stayed bit-identical to the detached path (max |Δρ_ion| = 0.000×10⁰) at every stage. Hover for values.

8Cost and status

Measured training cost (3×A100, fixsol=5): the backward adds five adjoint solves per structure (~100 PCG iterations each). Unused reduction levers: a single coupled adjoint at fixsol=5, looser adjoint tolerance, gradients on alternate refreshes only.
Scope by design: the gradient feeds the electrostatic observables (potential, Fermi, Φ1D). Energy and forces stay on the detached path — the adjoint must not enter the force graph (second-order autograd) — so E/F keep the lagged self-consistent treatment.

Available behind --solvent_pb_differentiable (default off, byte-identical). A training smoke ran 2,843 adjoint solves through DDP backward without incident. Whether the gradient improves the potential-family observables is an open A/B — deprioritized after the attribution study (report 2/3) showed the scalar potential converges equally well without it; the candidate benefit is confined to the Φ1D/Fermi gap (effect A).
