Where the 1-D potential error comes from

The PB run's 1-D potential profile error (Φ1D) plateaus about 0.05 eV above the old planar baseline. Five single-variable runs split that gap into three separate effects — none of them a bug in the PB integration.

1Five controlled runs

All runs: 120 epochs, identical data, loss weights, learning rate and seed; one variable changes at a time. Φ1D is scored against the model's actual solvent charge profile (an earlier gaussian-based reconstruction added a small ~0.006 eV metric artifact; planar layers are unaffected by construction).

2Effect A — the PB representation itself

≈ 0.029 eVPB vs planar, everything else identical

With code, sampler and supervision identical, PB sits above planar. Numerical probes rule out the integration machinery: for the same layer, the PB-path SCF features match the analytic planar path to ~10⁻⁵ V (against a 2.4 V atom-to-atom span), the 1-D compensation energy to 3×10⁻⁴, and the potential observable exactly.

The scalar potential is not the problem. A legacy run with a completely static solvent layer reaches potential RMSE 0.066 eV — the same as PB — showing that lagged self-consistency (layer recomputed from the current density, no gradient) is sufficient for scalar observables. The PB deficit is concentrated in the profile-shaped observable Φ1D and in the Fermi level.

Mechanism, current picture: the PB profile responds to the evolving density (electrostatic screening), so the residual density fits against a per-structure reference that keeps moving during training; a planar layer's geometry barely moves. Partially open — the differentiable-PB A/B (report 3/3) is the designed test.

3Effect B — the static-shard sampler

≈ 0.011 eVsampler on vs global shuffle, planar pair

Pinning each structure to one rank keeps the φ warm-start cache local (98.6% hit rate, ≈1.7× faster solve epochs, estimated) but reduces gradient diversity — each rank sees ⅓ of the data. The planar pair differing only in the sampler shows a consistent gap at every checkpoint from epoch 30 on.

Evidence level: a single-seed pair; not yet replicated, and not yet measured on PB directly. A rank-independent shared φ cache (designed to keep the speed without the sampler) is implemented and under validation.

4Effect C — the jellium feature convention

≈ 0.013 eVperiodic (corrected) vs isolated (pre-fix)

Switching the compensation-layer features back to the pre-fix “isolated” convention recovers Φ1D to the legacy level (0.062 vs 0.060) — but energy degrades from 2.9 to 5.3 meV/atom, and the convention reintroduces the spurious charged-cell parabola that the jellium fix removed, which breaks charge extrapolation. This is a physical-correctness vs training-set-fit trade-off, not a defect: production keeps periodic and accepts the Φ1D share.

5Fermi tracks Φ1D

The Fermi-level error orders the runs identically (0.120 / 0.074 / 0.063 / 0.052 eV at epoch 100) — one mechanism, not two. Anything that closes the Φ1D gap is expected to close the Fermi gap with it.

6Open items

  • Sampler effect measured on PB directly, and shared-cache validation (run in progress).
  • Whether effect A is reducible at all — the differentiable-PB A/B (infrastructure ready, report 3/3) is the designed experiment.
  • Single-seed caveat on effects B and C: magnitudes are ~2–5× the plausible run-to-run noise but have not been replicated.
